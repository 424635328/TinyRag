工作原理

从文档到回答的完整流程

01

加载文档

TextLoader 把知识文本统一为 Document 格式

02

切分文本

RecursiveCharacterTextSplitter 保留语义边界

03

生成向量

HuggingFaceEmbeddings 生成向量并交给 FAISS

04

检索相关

Retriever 只拿最相关片段，减少上下文开销

05

生成回答

LCEL 把 context、prompt 和 LLM 串成一条链

开始使用

立即体验智能知识库问答

快速开始

  1. 准备你的知识库文件（txt、md、pdf 等）
  2. 启动 Ollama 并拉取 qwen2.5:1.5b 模型
  3. 运行 `uvicorn web_app:app --reload`
  4. 打开浏览器访问 http://localhost:8000
  5. 点击"开始聊天"按钮开始使用
